# policy/

## Purpose
Defines policies for transaction relay, fee estimation, and mempool acceptance. Not consensus rules, but local node policies.

## Key Classes
- `PolicyEstimator`: Fee estimation
- `MempoolPolicy`: Mempool acceptance
